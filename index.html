Projects

Programming projects

GrowWithU Website

GrowWithU is a website created using HTML, CSS, Javascript and a little bit of PHP. Website is completely responsive, PHP was used so that administrators can edit, add, remove website testimonials.

Python script for TikTokLive

I made a python script for my friend who live streams on TikTok. Point of the script is simple, whenever someone gifts, follows, likes etc. the live creator, something can happen in game (do a 180, do an emote, Alt+F4...). It was my first attempt at using the TikTokLive github library, made to work from console.

Speech recognition project

My first attempt at Python speech recognition. When ever the user said skip, program would automatically press left mouse click. It was made to work from console.

TikTok Action Web project

TikTokLive Node.js project. I made a working website where user can select the event, what happens when event occurs and the amount of events required for action to take place.

Tikkit project

Tikkit application is an application created using C# .NET Framework, used for scraping TikTok streams, and performing certain actions when events occur. User can select the tiktok username, events, amounts of events and actions. It is still a work in progress, I still need to make the tables look nice...

Voice Control project

Voice Control project is a Python gui project. Idea is, you use the first input to select the word that is required for an action to occur, select the action from dropdown and when you press start, program listens for certain words and when it hears them, does a specific action.
